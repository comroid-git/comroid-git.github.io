The KScr intermediate language

GitHub Page

The KScr intermediate language aims at providing a very simplistic base syntax to aid with creating Streaming-API focused tasks. With its unique way of implementing Providers and Handlers besides Expressions and Statements, it makes creating streaming-tasks easier than ever before.



Syntax Elements

Variable Declaration

Every normal declaration, emitting and creating a value reference. The void type declares the absence of type in the reference.
// numeric variable declaration num x;
// long variable declaration num<long> x;
// string variable declaration str a;
// variable declaration without type void any;

Constant Expressions

Every literal symbol in KScr is compiled into a Constant Expression.
// int literal 5
// double literal 5.3d
// string literal "hello world"
// null literal null

Provider Expression

Basic exression components are compiled as Provider expressions. A provider expression may be used as a normal value expression.
// value cast (int) x
// member invoker txt.length
// variable accessor any

Pipe Declaration

Basic pipe variable declaration for chaining.
// pipe base node pipe<str> stringPipe;
// invokes the pipe with the expression value stringPipe <== expression;
// reads from the provider and invokes the pipe with the return value stringPipe <== provider;

Pipe Value Provider

Constructs a pipe that is reading from the provided handler and invokes after the handler.
// links the handler to the stringPipe stringPipe <== handler;

Pipe Value Handler

Applies a handler to a pipe. The hanlder is invoked every time the pipe is invoked.
// constructs an emitting pipe out of stringPipe stringPipe ==> lambda;
// SQL inspired filtering pipe operator stringPipe select(lambda: bool)
// SQL inspired reformatting pipe operator stringPipe where(lambda: TOut)

Lambda Pipe

A basic intermediate pipe operation that is used to produce output values.
// anonymous handler and reformatting Function // static function lambda Type::getTypes
// dynamic function lambda any::toString
// basic expression lambda it -> it.length
// statement lambda it -> { }
